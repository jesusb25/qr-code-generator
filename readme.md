# QR Code Generator

Turn any link into a scannable QR code in seconds.

## Live Website
[https://jesusb25.github.io/qr-code-generator/](https://jesusb25.github.io/qr-code-generator/)

## What It Does
- Generates a QR code from any URL you enter.
- Lets you download the generated QR code as a PNG file.
- Shows success/error messages for a smoother experience.
- Includes a fun entrance animation when the page loads.

## How To Use
1. Open the site.
2. Paste your link in the input box.
3. Click `Generate QR Code`.
4. Click `Save as PNG` to download it.

## Run Locally
1. Clone this repository.
2. Open `index.html` in your browser.
